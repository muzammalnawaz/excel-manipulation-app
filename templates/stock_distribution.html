Step 2: Configure Analysis Period

📅 Distribution Logic Overview

How the system works:

  1. Priority 1 - Existing Articles: High conversion items (≥80%) distributed to performing sites
  2. Priority 2 - Entirely New Articles: Articles not in any site's sales history
  3. Priority 3 - Site-wise New Articles: Articles new to specific sites but existing elsewhere
  4. 60-Day Rule: No distribution if current stock + recommended > 60 days supply
  5. Performance Priority: Sites with higher MC sales get preference

📊 Sales Analysis Period (for Distribution Logic)

📈 Analysis Logic: This period calculates daily sales averages for each MC at each site. The system uses this to determine stock days and distribution limits.
